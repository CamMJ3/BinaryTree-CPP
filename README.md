# Árbol Binario en C++

Implementación de un árbol binario en C++ con funciones de inserción, 
búsqueda, visualización y análisis estadístico de la información almacenada en la estructura.

## Características

- Inserción de nodos.
- Conteo de estructura.
- Análisis de datos.
- Visualización selectiva.
- Destructor recursivo.

## Estructura del proyecto
- doc/
    - html/
    - latex/

- images/
    - Imágenes de resultados de pruebas.

- src/
    - arbol.cpp
    - nodo.cpp
    - main.cpp

- headers/
    - arbol.h
    - nodo.h
    - excepcion.h

## Documentación
Toda la documentación ha sido generada con Doxygen.

Para visualizar: Diríjase a doc/html/index.html 
Posteriormente, abra con su navegador de preferencia.

Para visualizar todos los comentarios: Vaya a "Class Members", después,seleccione "All".

O abra el archivo PDF "refman.pdf".

## Ejecución
- Usando comandos: Abra su terminal, y ejecute el siguiente código...

g++ src/main.cpp -Iheaders -o binArbol
./binArbol
