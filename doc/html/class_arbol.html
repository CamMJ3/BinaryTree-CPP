◆ insertarDato()

template<typename T>
void Arbol< T >::insertarDato ( T dato)

Inserta un nuevo nodo con el valor de "dato" en el árbol.

El nuevo nodo se inserta en la "raiz" dado el caso que esta se encuentre vacia. De lo contrario, se insertará en el subárbol izquiero si el valor del nuevo nodo es inferior al de la raíz, o en el subárbol derecho si es superior.

Parameters
datoEl valor que se almacenará en el nuevo nodo del árbol.

template<typename T>
void Arbol< T >::visualizacion ( )

Imprime el contenido de los nodos por nivel.

Se organizan los nodos almacenados en el árbol dentro de una cola para mostrarlos por nivel; gracias al bucle while, primero imprime el número del nivel actual, seguido de los nodos y la categoría a la que pertenecen (raíz, hoja u interno). El uso de !cola.empty() asegura el no perder ningún nodo del árbol.

Dado el caso que el árbol se encuentre vacío, se lanza una excepción.
