\doxysubsection{Detailed Description}
Define la estructura del árbol binario. 

Su estructura consta de un puntero que referencia a la raíz del árbol, del cual se pueden derivar nodos internos y hojas.

Este archivo define los constructures y métodos de la plantilla \doxylink{class_arbol}{Arbol}.
