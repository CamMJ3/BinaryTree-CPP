\doxysubsection{Detailed Description}
El programa completo del árbol binario. 

Las opciones son las siguientes:\+
\begin{DoxyItemize}
\item Opción:\+ 1 Insertar un nodo del valor que el usuario desee.
\item Opción 2:\+ Mostrar el número total de nodos, y el número de nodos internos.
\item Opción 3:\+ Mostrar el promedio de los valores de los nodos, así como el valor máximo.
\item Opción 4:\+ Mostrar la categoría y valor de los nodos por nivel (de 0 en adelante).
\item Opción 5:\+ Mostrar el contenido del árbol en pre-\/orden, in-\/orden y post-\/orden.
\item Opción 6:\+ Salir del programa. 
\end{DoxyItemize}
